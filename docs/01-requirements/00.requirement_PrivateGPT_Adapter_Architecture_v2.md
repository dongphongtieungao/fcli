# PrivateGPT Adapter Server - Architecture & Technical Requirement Specification

Version: 2.0
Status: Published Architecture
Primary Mode: Agent Mode
Target Client: OpenCode CLI
Implementation Language: Python

## Parent Requirement

This published appendix defines the target architecture for the canonical [PrivateGPT Adapter requirement](./00.requirement.md).

---

## 1. Executive Summary

PrivateGPT Adapter là một dịch vụ Python chạy localhost, cung cấp OpenAI-compatible API cho các công cụ AI Coding Agent như OpenCode.

Adapter đóng vai trò cầu nối giữa OpenCode và hệ thống PrivateGPT nội bộ, đồng thời che giấu hoàn toàn chi tiết triển khai, cơ chế xác thực và API nội bộ của PrivateGPT.

Mục tiêu:

- Hỗ trợ OpenAI-compatible API.
- Hỗ trợ Agent Mode làm chế độ mặc định.
- Hỗ trợ browser-session authentication.
- Hỗ trợ streaming.
- Hỗ trợ mở rộng nhiều AI Provider trong tương lai.
- Không phụ thuộc LiteLLM.

## 5. Core Architectural Patterns

## Adapter Pattern

Tách biệt OpenAI API và PrivateGPT API.

## Provider Pattern

Cho phép thay đổi backend AI mà không ảnh hưởng client.

## Service Layer Pattern

Tách business logic khỏi API layer.

## Circuit Breaker Pattern

Ngăn backend bị quá tải.

## Retry Pattern

Xử lý lỗi tạm thời.

## Cache Pattern

Giảm số lần truy cập backend.

## 9. HTTP Client Architecture

Dùng singleton AsyncClient.

```python
httpx.AsyncClient(
    timeout=120,
    limits=httpx.Limits(
        max_connections=200,
        max_keepalive_connections=50
    )
)
```

Nguyên tắc:

- Connection pooling.
- HTTP keepalive.
- Async only.

## 14. Security Requirements

- Bind localhost mặc định.
- Không hardcode secret.
- Secret lưu trong .env.
- Không log token.
- Không log cookie.
- Validate mọi request.
- Rate limiting bắt buộc.
