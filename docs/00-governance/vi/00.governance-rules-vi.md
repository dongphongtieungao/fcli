# Quy Tac Governance

## Purpose

Xac dinh cac quy tac governance co the tai su dung cho starter repository.

## Context

Cac quy tac nay duoc giu o muc generic. Chi them rule ID rieng cho project khi scope moi da ro.

## Rules

| ID | Quy tac | Muc do | Cach kiem tra |
|---|---|---|---|
| GOV-001 | Tai lieu official khong duoc tu tao bang chung source-of-truth. | Major | Documentation review |
| GOV-002 | Khong commit secrets hoac du lieu nhay cam. | Blocker | Review hoac secret scan |
| GOV-003 | Thay doi hanh vi nen co tests hoac verification notes tuong ung. | Major | Test/evidence review |
| GOV-004 | Tai lieu ben vung moi nen duoc lien ket voi artifact lien quan. | Minor | Link review |
| GOV-005 | Workspace notes khong duoc override official docs. | Major | KMS review |
