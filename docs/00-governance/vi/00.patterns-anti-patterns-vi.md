# Patterns Va Anti-Patterns

## Purpose

Ghi lai cac pattern ky thuat va tai lieu co the tai su dung cho starter repository.

## Context

Pattern rieng cua project nen duoc them sau khi project moi co module, contract va workflow that.

## Patterns

| Pattern | Dung khi | Loi ich |
|---|---|---|
| Spec before implementation | Hanh vi anh huong nguoi dung hoac van hanh | Giam mo ho |
| Small scoped changes | Task cham vao file dung chung | Giam rui ro regression |
| Source-linked docs | Tai lieu dieu khien code hoac verification | Tang traceability |
| Workspace before promotion | Phat hien chua phai official | Giu docs sach |

## Anti-Patterns

| Anti-pattern | Rui ro | Cach an toan hon |
|---|---|---|
| Xoa theo keyword | Mat scaffold co the tai su dung | Phan loai theo vai tro truoc |
| Xem workspace notes la official | Tao drift | Promote bang evidence |
| Claim verification khi chua chay | Gay hieu nham | Bao cao partial hoac pending checks |
| Tao orphan docs | Kho bao tri | Them related links that |
