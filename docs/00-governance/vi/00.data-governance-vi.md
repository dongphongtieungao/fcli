# Quan Tri Du Lieu

## Purpose

Ghi lai khung quan tri du lieu co the tai su dung cho project moi.

## Context

Project hien tai chua co mien nghiep vu cu the. Tai lieu nay giu cac cau hoi va nguyen tac nen duoc tra loi khi project bat dau co nguon du lieu that.

## Governance Baseline

| Khu vuc | Nguyen tac khoi diem | Bang chung can bo sung sau |
|---|---|---|
| So huu du lieu | Moi tap du lieu ben vung nen co owner hoac `unknown`. | Danh sach owner hoac RACI |
| Phan loai | Can biet muc do nhay cam truoc khi chot quy tac luu tru va logging. | Classification matrix |
| Luu giu | Retention va deletion rule cho artifact sinh ra phai ro rang. | Retention policy |
| Chat luong | Data quality check nen gan voi acceptance criteria. | QA matrix hoac tests |
| Truy cap | Least privilege la mac dinh. | Access model |
| Auditability | Cac transformation quan trong nen co the tai lap. | Runbook hoac verification notes |

## Open Questions

- Project moi se dung nguon du lieu nao?
- Du lieu nao la sensitive, regulated, hoac customer-owned?
- Artifact nao can retention hoac secure deletion rule?
