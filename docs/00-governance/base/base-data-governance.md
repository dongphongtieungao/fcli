# Base Data Governance

## Purpose

Provide a reusable baseline for data governance documents.

## Context

Use this as a neutral starting point before adding project-specific data sources, compliance requirements, and retention rules.

## Baseline Questions

- What data is collected?
- Who owns it?
- Where is it stored?
- How is it validated?
- How long is it retained?
- Who can access it?
- How are transformations audited?
